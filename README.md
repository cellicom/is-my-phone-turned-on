# Is My Phone Turned On?

A minimalist and lightweight Android application designed to answer the fundamental question: "Is my phone turned on?".

> [!NOTE]
> This is a **prank app** created for **educational purposes** as a training exercise for using **traditional XML Views** in Android, ensuring maximum lightness and compatibility.

## Features

- **Minimal Interface**: A single screen focused on the essentials.
- **"Loading" Process**: Upon launch, the app shows a loader with the text "Loading" for 3.5 seconds, simulating a complex analysis.
- **Definitive Answer**: After the wait, a large **"YES"** appears in the center of the screen.
- **Exit Confirmation**: Advanced back button handling with a confirmation modal to prevent accidental exits.

## Technical Specifications

- **Language**: Kotlin
- **UI Framework**: Traditional Android XML Views (Zero Jetpack Compose runtime overhead).
- **Package Name**: `com.ismyphoneturnedon.cellicomsoft`
- **Version**: `2.0.1`
- **Target SDK**: 35 (Android 15)
- **Min SDK**: 24 (Android 7.0)

## Project Purpose

The project demonstrates how to create a modern app that meets Google Play Store requirements while maintaining an infinitesimal weight, avoiding heavy libraries, and using the operating system's native components.
